This is a list of resources I found useful while learning the
[Rust](https://rust-lang.org/) programming language.

## Introductory

I recommend starting by reading [The Rust Programming
Language](https://doc.rust-lang.org/book/) (also available [as a
printed book](https://nostarch.com/rust-programming-language-3e)).

[Programmin Rust (O'Reilly)](https://www.oreilly.com/library/view/programming-rust-2nd/9781492052586/)
is also a great entry-level book, which gets a bit more into detail,
especially about concurrency and async programming. I recommend
reading it after the official Rust Book.

## Real-world Rust

The next book I recommend is
[Effective Rust](https://effective-rust.com/), available both online and as a
printed book.
It gets into more detail about using Rust in the real world.

Other resources I recommend:

- [Rust Design Patterns](https://rust-unofficial.github.io/patterns/)
- [Rust API guidelines (official)](https://rust-lang.github.io/api-guidelines/about.html)
  -- especially useful when creating libraries that others will use;
  contains a lot of useful guidelines about best practices.
- [Awesome Rust](https://github.com/rust-unofficial/awesome-rust) -- a collection of Rust code and resources.
